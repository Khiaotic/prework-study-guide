HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • HTML attributes are always defined in the beginning element never the closing element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A link element ccreates the connection to the external file
  • The rel attributespecifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location or URL of the external resource.
  • Declaration contain two important components: the CSS property we want to apply and the value of the property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd and mkdir is used to navigate and create directories or perfom other tasks
  • md or markdown is a way to write in Github that converts plain text to Github code
  • commit changes are a saved record of change made to a file within the repo

JavaScript

  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way
  • Control flow is the order in which a computer executes code in a script
  • Commenting out code makes it inactive without deleting it, just in case you need it later
